Api

URL: http://158.39.188.204/steg1/api

Register a new user

/auth/register

Request parameters

{ "first_name": "example", "last_name": "example", "email": "email@example.com", "password": "example", "role": "student" }

Responses

{ "success": true, "message": "User registered successfully.", "data": { "user_id": 123, "email": "john.doe@example.com", "role": "student" } }
{ "success": false, "message": "Validation error: Missing or invalid fields.", "data": { "missing_fields": [ "email" ] } }

Login a user

/auth/login

Request parameters

{ "email": "email@example.com", "password": "example" }

Responses

{ "success": true, "message": "Login successful.", "data": { "user_id": 123, "role": "student" } }
{ "success": false, "message": "Invalid email or password.", "data": {} }

Logout the current user

/auth/logout

Responses

{ "success": true, "message": "Logout successful.", "data": {} }
{ "success": false, "message": "Not logged in.", "data": {} }

Change the user’s password

/auth/change-password

Request parameters

{ "current_password": "example", "new_password": "example", "confirm_password": "example" }

Responses

{ "success": true, "message": "Password changed successfully.", "data": {} }
{ "success": false, "message": "Validation error: Passwords do not match.", "data": {} }
{ "success": false, "message": "Unauthorized: You must be logged in.", "data": {} }

Request a password reset link

/auth/password-reset/request

Sends an email containing reset instructions if the user exists.

Request parameters

{ "email": "email@example.com" }

Responses

{ "success": true, "message": "Password reset instructions sent to your email.", "data": {} }
{ "success": false, "message": "Validation error: Email is missing or invalid.", "data": {} }
{ "success": false, "message": "Email not found.", "data": {} }
